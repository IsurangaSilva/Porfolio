# portfolio-nextjs

Opinionated starter portfolio built with Next.js (App Router) + TypeScript.

Features included:
- TypeScript + App Router
- Tailwind CSS with dark mode
- Framer Motion animations
- Simple MDX-ready blog folder (one example post)
- SEO helper, sitemap & robots placeholders
- Contact form integration notes (Formspree)
- Jest + React Testing Library sample test
- ESLint + Prettier configs

Quick start (Windows PowerShell):

```powershell
cd "c:/Users/Isuranga/Desktop/New folder/portfolio-nextjs"
npm install
npm run dev
```

Build for production:

```powershell
npm run build
npm start
```

Notes:
- Replace placeholder content in `app/` and `posts/` with your real projects and posts.
- Add analytics IDs in `components/Analytics.tsx` or the provided placeholder.
